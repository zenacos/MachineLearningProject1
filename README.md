Car Price Prediction: Linear Regression from Scratch
This project aims to predict car selling prices based on various features such as year, mileage, fuel type, and transmission. The core highlight of this repository is the custom implementation of the Linear Regression algorithm and Gradient Descent optimization without relying on high-level machine learning libraries like Scikit-Learn.

Key Highlights
From-Scratch Implementation: Built the entire Linear Regression model and Gradient Descent optimizer from the ground up using mathematical principles.
Manual Data Preprocessing: Performed data cleaning, manual mapping for ordinal data, and One-Hot Encoding for categorical features.
Feature Scaling: Implemented standardization (Z-score normalization) to ensure efficient convergence during the training process.
Hyperparameter Analysis: Conducted experiments with multiple learning rates to visualize their impact on the cost function and model convergence.

Evaluation Metrics
To measure the performance of the model, I used standard regression metrics:
MSE (Mean Squared Error)
RMSE (Root Mean Squared Error)
MAE (Mean Absolute Error)

Tech Stack
Python: Core programming language.
NumPy: Used for matrix operations and mathematical calculations.
Pandas: Used for data manipulation and preprocessing.
Matplotlib: Used for visualizing the learning curves and model performance.
